# Design System

Welcome to our comprehensive design system built with React, SCSS, and Storybook. This system provides a robust foundation for building consistent, accessible, and scalable user interfaces.

## 🎯 Overview

Our design system follows the Atomic Design methodology, providing a clear hierarchy of components from basic building blocks (atoms) to complex compositions (molecules). The system is designed to be:

- **Consistent**: All components share the same design tokens and behavior patterns
- **Accessible**: Built with accessibility in mind, following WCAG guidelines
- **Themeable**: Supports light and dark themes with automatic system preference detection
- **Scalable**: Easy to extend and customize for different use cases
- **Well-documented**: Comprehensive Storybook documentation with live examples

## 🏗️ Architecture

### Component Hierarchy

```
Atoms (Basic Building Blocks)
├── Button
├── Input
├── Typography
├── Icon
├── Badge
├── Avatar
├── Checkbox
├── Radio
├── Switch
└── TextArea

Molecules (Composed Components)
├── FormField (uses Input, TextArea, Typography)
├── Card (uses Typography, Button, Badge, Avatar, Icon)
├── Modal (uses Typography, Button)
├── Tabs (uses Typography, Button)
├── Accordion (uses Typography, Button)
└── Select (uses Typography, Input)
```

### Design Tokens

Our design system is built on a comprehensive set of design tokens that ensure consistency across all components:

- **Colors**: Semantic color system with light/dark theme support
- **Typography**: Consistent font families, sizes, and weights
- **Spacing**: Standardized spacing scale for margins and padding
- **Border Radius**: Consistent corner rounding values
- **Shadows**: Elevation system for depth and hierarchy
- **Breakpoints**: Responsive design breakpoints

## 🚀 Getting Started

### Installation

```bash
npm install
npm run dev
```

### Basic Usage

```jsx
import { 
  Button, 
  Input, 
  FormField, 
  Card, 
  Typography 
} from './src';

function MyComponent() {
  return (
    <Card variant="outlined">
      <Card.Header title="User Profile" />
      <Card.Body>
        <FormField
          label="Name"
          required
          inputType="input"
          inputProps={{ placeholder: 'Enter your name' }}
        />
      </Card.Body>
      <Card.Footer
        actions={<Button variant="primary">Save</Button>}
      />
    </Card>
  );
}
```

## 🎨 Theming

The design system supports both light and dark themes with automatic system preference detection:

```jsx
import { ThemeProvider, ThemeContextProvider } from './src';

function App() {
  return (
    <ThemeContextProvider>
      <ThemeProvider>
        <YourApp />
      </ThemeProvider>
    </ThemeContextProvider>
  );
}
```

## 🌟 Features

### Atom-Molecule Integration

Our molecules seamlessly integrate with atoms, providing:

- **Consistent Theming**: All components use the same design tokens
- **Composable Architecture**: Mix and match atoms to create molecules
- **Accessibility First**: Built-in ARIA support and keyboard navigation
- **Responsive Design**: Mobile-first approach with breakpoint tokens
- **Type Safety**: Full TypeScript support (coming soon)

### Storybook Integration

- **Auto-generated Docs**: Automatic documentation from component props
- **Interactive Controls**: Live editing of component properties
- **Accessibility Testing**: Built-in accessibility testing
- **Visual Testing**: Chromatic integration for visual regression testing
